How to use the scheduler:

To schedule an update, simply click on the start date on the calendar and select the months to update on, as well as the time and start date if you wish to specify this, and the days on which to update. For example, if you select “Jan” and “Mar” as the months, and “5” and “21” as the days, the scraper will run on January 5th, January 21st, March 5th, and March 21st.

The time is the time when the scraper will run on the selected dates, it can be left blank to start near midnight. If the time is left blank, the scheduler will intentionally wait a random amount of time (less than one minute) before scraping, in order to prevent the server from being overloaded. If the time is defined, the scraper will start at exactly that time.

The scraper will start on or after the specified start date if it is specified. The start date can be left blank if you want the scraper to start on the next selected day.

How to use manual entry:

In addition to scraping other websites, you can also manually enter the faculty information of individuals in specific fields, such as “name”, “profile”, “image”, “location”, “phone number”, etc.
